Key Concepts

  • HTML stands for HyperText Markup Language, it is the standard markup language for creating web pages, it was invented by Tim Berners-Lee in 1991. It has elements like headings, images and links and it works by using tags.
  • CSS stands for Cascading Style Sheets, it is a style sheet language used for styling web pages, it was invented by Hakon Wium Lie in 1996. It allows you to control the layout and appearance of web pages, it works by using selectors and properties.
  • JavaScript is a programming language that is used to make web pages interactive, it was invented by Brendan Eich in 1995. It allows you to add dynamic behavior to web pages, it works by using functions and events.
  • SGML stands for Standard Generalized Markup Language, it is a standard for defining markup languages. It was invented by James Clark in 1986.
  • W3C stands for World Wide Web Consortium, it is an international standards organization that develops web standards.
