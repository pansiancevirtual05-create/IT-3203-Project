Key Concepts

Concept Inventor / Organization Year Description
HTML Tim Berners-Lee 1991 HyperText Markup Language. Standard markup language for creating web pages.
CSS Hakon Wium Lie 1996 Cascading Style Sheets. Used for controlling the layout and appearance of web pages.
JavaScript Brendan Eich 1995 Programming language used to add dynamic behavior and interactivity to web pages.
SGML James Clark 1986 Standard Generalized Markup Language. A standard for defining markup languages.
W3C World Wide Web Consortium 1994 International standards organization that develops and manages web standards.
